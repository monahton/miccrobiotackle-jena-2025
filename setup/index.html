For Windows users

I. Installing R and RStudio

Install R from this .exe file on CRAN. Then install RStudio IDE.
If you have separate admin/user accounts, run the installer as administrator.

Video Tutorial

II. Installing Bash (Git Bash)

Bash is provided via Git Bash on Windows. Download and install Git for Windows using the following link https://gitforwindows.org/. The link will download an .exe file of teh setup to install Git Bash.

Video Tutorial

III. Installing Git

Git should be installed on your computer as part of your Bash install (see the Shell installation instructions above).

For Mac users

I. Installing R and RStudio

Download and install R from this .pkg file on CRAN. Then install RStudio IDE.

Video Tutorial

II. Accessing Bash

On macOS, you access Bash from the Terminal app located in /Applications/Utilities.

III. Installing Git

Open the Terminal app, type git --version and press Enter. If Git is not installed, follow the prompt to install the command line developer tools. Do not click "Get Xcode" — it’s not required.

For older macOS (10.5–10.8), use the “snow-leopard” installer from here. This project is no longer maintained.
If the installer is blocked, right click the file, select “Open”, then confirm again to install.

Video Tutorial

For Linux users

I. Installing R and RStudio

Follow the instructions at this CRAN Linux page for your distribution. For example, on Fedora run sudo dnf install R, or on Ubuntu add a PPA and run sudo apt-get install r-base.
Then download and install the RStudio IDE.

II. Accessing Bash

Bash is the default shell on most Linux distributions and can be accessed through your terminal emulator.

III. Installing Git

Git is often pre-installed. You can check by running git --version. If it's not installed, use your package manager:
For Debian/Ubuntu: sudo apt-get install git
For Fedora: sudo dnf install git
For Arch: sudo pacman -S git

For more information or if you encounter any trouble/errors in the installation process, please send an email to aboualezz.monah@hsr.it
