Installing R and RStudio

R is a programming language that is especially powerful for data exploration, visualization, and statistical analysis. To interact with R, we use RStudio.

For Windows users:

Install R by downloading and running this .exe file from CRAN. Also, please install the RStudio IDE.
Note that if you have separate user and admin accounts, you should run the installers as administrator (right-click on .exe file and select "Run as administrator" instead of double-clicking). Otherwise problems may occur later, for example when installing R packages.

For Mac users:

Install R by downloading and running this .pkg file from CRAN. Also, please install the RStudio IDE.

Video Tutorial


For Linux users:

Instructions for R installation on various Linux platforms (debian, fedora, redhat, and ubuntu) can be found at this link depending on the distribution. These will instruct you to use your package manager (e.g. for Fedora run sudo dnf install R and for Debian/Ubuntu, add a ppa repository and then run sudo apt-get install r-base).
Also, please install the RStudio IDE.


For more information or if you encounter any trouble/errors in the installation process, please send an email to aboualezz.monah@hsr.it
