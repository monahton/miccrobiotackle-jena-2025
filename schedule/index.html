Day 1 Program

8:30am Registration and Welcoming!
Participants’ registration and welcoming to the start of the workshop
We will also use this time to make sure all participants have the required software installed and to fix any installtions or setup issues.
9:00am Start - Introduction to the Workshop
Monah Abou Alezz
Introducing the instructors, outlining the code of conduct, and providing an overview of the workshop program.
9:30am Intoduction to the Unix Shell
A beginner-friendly introduction to key principles, including linux theory, perceptual science, and best practices.
10:15am Working with Files and Directories
Overview to the Unix commands and concepts. What are the different ways of navigating the files and folder structure in an operating system.
11:00am Coffee & Tea Break
11:20am Conditional Programming and Shell Scripting
Hands-on training focusing on the foundational aspects of creating basic loops, pipes and conditional statements in Unix.
1:00pm Lunch
2:00pm Introduction to R and Rstudio
A beginner-friendly introduction to key principles of R programming.
3:00pm Exercises and Applications
Interactive exercises that apply the basic techniques learned throughout the day.
3:30pm Coffee & Tea Break
3:50pm Exploring and Understanding Data in R
Explore and understand the charactersitcs of different basic data types and data structures in R.
5:00pm Finish Day 1

Day 2 Program

9:00am Recap session
Monah Abou Alezz
A brief recap of Day 1: a quick recap and Q&A session addressing participant questions and solving any problems encountered in the previous session.
9:30am Loading and Analysing Biological Data in R
Explore advanced techniques for customizing biological data analysis, focusing on improving reproducibility and adaptability to different datasets.
10:15am Exercises
Hands-on practice to apply data analysis concepts and customization techniques covered earlier in the session.
11:00am Coffee & Tea Break
11:20am Advanced Programming Techniques in R
Further delve into advanced strategies for complex data analysis and creating custom functions.
1:00pm Lunch Break
2:00pm Version Control with Git
Understand the benefits of an automated version control system and how it works
2:30pm Exercises and Applications
Engage in practical exercises to reinforce the concepts of version control applying them to real-world examples.
2:50pm Coffee & Tea Break
3:00pm Wrap-up
Summarize the key takeaways from the workshop, followed by a Q&A session and discussion on the next steps and post-workshop survey.
4:00pm Finish Day 2
