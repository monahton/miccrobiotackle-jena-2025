Lessons

This is a hands-on tutorial in which the data and code will be distributed to participants who wish to follow along. All tutorial lessons and materials will be provided and will be freely accessedat any time.

  • - Introduction to data visualization and dataset: https://monahton.github.io/mastering_data_visualisation/lessons/01_Introduction.html/
  • - Basic ggplot2 syntax and creating a basic plot: https://monahton.github.io/mastering_data_visualisation/lessons/02_ggplot2_syntax.html
  • - Consistent plots with custom themes: https://monahton.github.io/mastering_data_visualisation/lessons/03_custom_themes.html
  • - Application of plotting basics: https://monahton.github.io/mastering_data_visualisation/lessons/04_boxplot_application_of_basic_plotting.html
  • - Customization of scales and color palettes: https://monahton.github.io/mastering_data_visualisation/lessons/05_custom_plot_scales_colors.html
  • - Aligning and labeling plots with cowplot: https://monahton.github.io/mastering_data_visualisation/lessons/06_aligning_plots_using_cowplot.html
  • - Adding annotations and including statistical comparisons with ggpubr: https://monahton.github.io/mastering_data_visualisation/lessons/07_adding_text_annotations.html
  • - Incorporating external packages to extend plotting functionality: https://monahton.github.io/mastering_data_visualisation/lessons/08_figure_specific_packages.html
  • - Creating final figure: https://monahton.github.io/mastering_data_visualisation/lessons/09_final_figure.html
  • - Pivoting publications: https://monahton.github.io/mastering_data_visualisation/lessons/10_pivoting_publications.html
